# React Converted Fragment

Questa cartella contiene il progetto React generato dall'estensione.

## Avvio rapido (Vite)

1. Installare dipendenze

```bash
npm install
```

2. Avviare in sviluppo

```bash
npm run dev
```

3. Creare `index.html` con il root element

```html
<!doctype html>
<html>
  <head><meta charset="utf-8"/><title>Preview</title></head>
  <body>
    <div id="root"></div>
    <script type="module" src="/index.js"></script>
  </body>
</html>
```

Nota: gli stili sono in `styles.css`. I componenti sono in `components/`.
